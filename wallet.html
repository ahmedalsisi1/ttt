A secure home for your assets. Send, receive and hold with confidence.

  • Easy transfers and transaction history.
  • Ownership remains with the user.

Developer value

  • Standard signing flows for dApps.
  • Docs and code examples.

Key features

  • Account creation and backup guidance.
  • Multi-asset foundation for future tokens.
