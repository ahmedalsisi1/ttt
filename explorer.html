Open Explorer

TikkoX Explorer

See your transactions, track blocks and accounts, and verify activity in real time.

Purpose

The Explorer lets anyone search blocks, transactions and accounts instantly. It’s the public window into the network, built for users and analysts alike.

User value

  • Track transfers and confirmations.
  • Search by block / tx / account from one place.
  • See live status without installing tools.

Developer value

  • Debug apps by checking on-chain state quickly.
  • WebSocket-powered live updates.
  • Reference endpoints listed in docs.

Key features

  • Global search (block / tx / account).
  • Live counters: blocks, peers, runtime info.
  • Token view with internal pricing.
